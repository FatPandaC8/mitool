# Usage
go install github.com/FatPandaC8/mitool@latest

2 main features: SSH accounts and generating Golang folder structure based on templates

Suggestions: 
- dotfile manager (like GNU stow)
- doctor to check any services is down or something went wrong
- projects management using sqlite & find 

mitool init
- Create config directory
- Create database
- Run migrations
- Prepare default directories
- Be safe to run multiple times
